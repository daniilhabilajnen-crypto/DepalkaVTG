# Telegram Points Bot PostgreSQL v8

- Случайные ежедневные задания с наградой баллами и XP.
- Сброс бонуса, заданий и лимита подадмина в 00:00 по времени Финляндии.
- Все игры доступны с 1 уровня.
- Подадмины: лимит выдачи 1000 баллов в сутки, бонус 1000, ×1.5 XP.
- Ивент `золотой дождь`.
- Режим `мега удача 100`.
- Магазин и админская выдача бустеров.

Замените в GitHub `bot.py` и `requirements.txt`.
