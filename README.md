
# Telegram Points Bot PostgreSQL v7

Полностью новая версия бота на PostgreSQL.

## Переменные Railway

В сервисе бота должны быть:

```env
BOT_TOKEN=токен_бота
DATABASE_URL=${{Postgres.DATABASE_URL}}
```

## Обновление GitHub

Замените:

- `bot.py`
- `requirements.txt`

После Commit changes Railway автоматически выполнит новый деплой.

## Важно о старых данных

SQLite-балансы автоматически не переносятся. После первого запуска PostgreSQL будет пустым, и каждый новый пользователь получит 2500 баллов.

## Админ

Единственный администратор:

```text
@Some_RaNdOmuser
```

Команды по ответу на сообщение игрока:

```text
выдать 500
снять 500
обнулить

выдать уровень 3
забрать уровень 2
установить уровень 10
```

При ручном изменении уровня опыт игрока сбрасывается до 0.

## Уровни

- Уровень 1: Мины
- Уровень 2: Джокер
- Уровень 3: Кости
- Уровень 4: Монета
- Уровень 5: Рулетка
- Уровень 6: Дуэль

Максимальная ставка:

```text
1000 + 250 × (уровень - 1)
```
